About the Auto Grader

I’ve implemented an auto grader for the class. This is the first semester it’s been used, and the first time in a statistics class as far as I know, so it is still in beta. The format of providing your answers is different than before, so I’ve provided some examples below.

1. Simple Example Question

In the case of a question with a direct answer, you’ll assign your answer to a variable named after the question number (in this case, answer1).

Obtain the Star Wars characters whose height is greater than 100 cm using the data frame starwars.

answer1 <- NULL # replace with your answer
answer1 <- dplyr::filter(dplyr::starwars, height > 100) # what you'd write

1.1. Example Question with Subparts

In cases of questions with subparts, you may be asked to assign your answer to items in a list. Use the $ operator to access and set the answers.

answer1.1 <- list(
  a = NULL,
  b = NULL,
  c = NULL
)

1.1A: Numeric Persona

What is your favorite number?

answer1.1$a <- 7

1.1B: A Lovely Sentence

What is your favorite sentence?

answer1.1$b <- "GO BEARS!"

1.2C: Animalistic

What is your favorite animal?

answer1.1$c <- "bear"

1.3: Example Question with a Function

In other cases, you’ll be asked to write a function. In that case, just fill in the body of the function.

Write a function every_nth that obtains every \(nth\) element from a vector for \(n \geq 1\).

every_nth <- function(v, n) {
  # Your code here
  v[c(TRUE, rep(FALSE, n - 1))] # error checking excluded
}

Uh-oh, LeetCode in R?

Code 
two_sum <- function(nums, target) {
  num_indices <- new.env() # we use an environment as a hash map
  for (i in seq_along(nums)) {
    dif <- as.character(target - nums[i])
    dif_index <- num_indices[[dif]]
    if (!is.null(dif_index))
      return(c(i, dif_index))
    num_indices[[as.character(nums[i])]] <- i
  }
  -1
}
